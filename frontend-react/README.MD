# Frontend (React)

This directory is reserved for the React-based frontend layer.

## Purpose
The frontend will consume the Laravel Articles API (`/api/articles`)
and render blog content for users.

## Planned Responsibilities
- Fetch paginated articles from backend API
- Display article list and detail views
- Handle basic loading and error states

## Status
Not implemented as part of this assignment.

The primary evaluation focus is:
- Backend scraping correctness
- API design
- LLM integration readiness

This folder exists to demonstrate planned architecture
and clean separation of concerns.
